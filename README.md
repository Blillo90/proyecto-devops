# Proyecto fin de grado ASIR - DevOps - Plataforma web con Kubernetes + Ansible

Este repositorio contiene la Infraestructura como Código (IaC), manifiestos Kubernetes y documentación de mi proyecto de fin de ciclo ASIR.

## Contenidos

- 'ansible/' : Playbooks para instalar y configurar el cluster K3s.
- 'k8s/' : Manifiestos Kubernetes para desplegar la aplicación web por completo.
- 'docs/' : Documentación técnica del proyecto
